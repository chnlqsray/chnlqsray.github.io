半导体薄膜全自动计量与 AI 分析平台（GaN PL 项目） Automated Semiconductor Metrology & AI Analysis Platform (GaN PL)
2024/09 — 至今  ·  核心研发  ·  曼彻斯特大学 Sep 2024 – Present  ·  Core R&D  ·  The University of Manchester
⬡ 论文撰写中 ⬡ Manuscript in Preparation

围绕博士课题，从零构建集光路设计、硬件控制、视觉反馈与光谱采集于一体的全自动 GaN 光致发光计量平台，覆盖从物理机理到系统交付的完整链路。 Built a fully automated GaN photoluminescence metrology platform from scratch for my PhD research, covering the full chain from optical-path design and hardware control to radiometric calibration and AI-enhanced analysis.

  • 系统架构：从零构建 GaN 光致发光计量平台，集光路设计、硬件控制（相机/压电台/光谱仪）、视觉反馈与光谱采集于一体；主导反射镜、带通滤光片、分束镜等核心器件选型与链路优化；利用 Python SDK 实现仪器深度集成 System architecture: Built end-to-end automated GaN PL metrology platform combining optical-path design, hardware control (camera, piezo stages, spectrometer), vision feedback and spectral acquisition; led component selection (mirrors, bandpass filters, beam splitters) and optical-path optimisation; instruments integrated via Python-based SDK control
  • 视觉驱动对焦：基于计算机视觉设计自动对焦算法，采用 Otsu 分割 + 爬山搜索，以光斑面积比例为目标函数，实现亚微米级焦平面追踪 Vision-based autofocus: Designed computer-vision workflow using Otsu segmentation + hill-climbing search with beam spot-area ratio as objective function, achieving sub-micron focal-plane tracking
  • 辐射定标：建立白光源定标流程与系统光谱非线性补偿模型；通过 SPF2 字节级逆向解析读取曝光时间，集成坏点修复与批处理链路，确保跨波段量值溯源性 Radiometric calibration: White-light-source calibration and non-linear spectral response compensation; dead-pixel correction via SPF2 byte-level exposure-time extraction; batch processing for cross-band physical traceability
  • AI 增强分析层：通过 AI 协作交付约 6200 行 Python 的 Streamlit 平台——多峰拟合、功率依赖寿命分析、速率方程建模（differential evolution）、CIE 色坐标、二维 Mapping 热图与 AI Copilot 写作辅助；同时构建本地 RAG 文献问答助手，支持将科研文献向量化后持久化检索，辅助文献调研与写作 AI-enhanced analysis layer: Streamlit platform (~6,200 Python lines) via AI collaboration — multi-peak fitting, power-dependent lifetime analysis, rate-equation modelling (differential evolution), CIE chromaticity, 2D mapping and AI Copilot writing assistance; additionally built a local RAG literature Q&A assistant enabling persistent vector retrieval of research papers to support literature review and writing

光学器件表征与系统不确定度分析（微环谐振器项目） Optical Module Characterisation & System Uncertainty Analysis (Microring Resonator)
2022/09 — 2024/07  ·  核心研发  ·  曼彻斯特大学 Sep 2022 – Jul 2024  ·  Core R&D  ·  The University of Manchester
● 已发表 ● Published

微环谐振器光纤耦合表征：建立六自由度对准协议与测量系统分析框架，通过 COMSOL 数字孪生验证加工公差对光栅耦合器耦合效率的影响；非标夹具设计将硬件成本降低 90% 以上。 Microring resonator fibre-to-waveguide characterisation: established a six-degree-of-freedom alignment protocol and measurement system analysis framework; root cause analysis of fabrication-tolerance effects on grating coupler's coupling efficiency via COMSOL digital-twin modelling.

  • 精密对准协议：基于 Thorlabs 1550 nm SLD 宽谱光源与压电控制系统，针对光纤–波导耦合建立高效稳定对准流程，实现亚微米级链路调节 Precision alignment: 6-DoF fibre-to-waveguide alignment using Thorlabs 1550 nm SLD broadband source + piezo controllers, enabling sub-micron link adjustment
  • 测量系统分析（MSA）：针对环境振动与热漂移设计重复性与稳定性测试，对质量因子 Q、消光比等关键指标进行多样本统计分析，验证统计显著性 Measurement system analysis (MSA): Repeatability and stability studies quantifying Q-factor and extinction ratio variation under environmental vibration and thermal drift; statistical validity ensured
  • 仿真一致性验证与根因分析（RCA）：利用 COMSOL 建立数字孪生模型，对比实验与仿真数据，对光栅耦合器耦合效率进行加工公差根因分析 Simulation & root cause analysis (RCA): COMSOL digital-twin models; RCA of fabrication-tolerance effects on grating coupler's coupling efficiency, correlating simulation with experimental results
  • 快速原型设计：利用 Fusion 360 与 3D 打印技术设计非标光纤夹具，在保证数百微米级定位精度的前提下将硬件成本降低 90% 以上 Rapid prototyping: Non-standard fibre holders via Fusion 360 + 3D printing; hundreds-of-microns positioning accuracy; hardware cost reduced by >90%

基础设施 · 自动化运维 Infrastructure · Automated Operations

支撑以上所有部署应用持续运行的后台系统，每日自动推送个人投资数据的定时任务，以及基于 Playwright 的 PDF 导出脚本。代码分别托管于 finance_app_cloud、etf_daily_report 与 chnlqsray.github.io 仓库。 Backend systems keeping all deployed applications running continuously, a scheduled task that auto-delivers daily personal investment data, and a Playwright-based PDF export script. Code hosted in the finance_app_cloud, etf_daily_report, and chnlqsray.github.io repositories.

应用保活系统 Keep-Alive System

GitHub Actions 每 6 小时触发，Playwright 无头浏览器模拟真实用户访问，自动检测并点击 Streamlit 休眠唤醒按钮，含截图存档，确保两款应用全天候在线。 GitHub Actions triggers every 6 hours; Playwright headless browser simulates real user visits, automatically detects and clicks the Streamlit wake-up button, with screenshot archiving to ensure both apps remain online around the clock.

ETF 每日日报 ETF Daily Report

工作日定时执行，调用腾讯行情 API 拉取 A 股数据（解决 GitHub Actions 海外服务器跨境限制），yfinance 拉取美股数据，计算五期收益率，生成带颜色标注的 HTML 邮件每日推送。 Runs on a weekday schedule, fetching A-share data via Tencent Market API (bypassing GitHub Actions cross-border restrictions) and US equity data via yfinance, computing five-period returns and delivering a colour-coded HTML email daily.

作品集 PDF 导出 Portfolio PDF Export

复用同一 Playwright 环境构建脚本化 PDF 导出方案，强制 printBackground: true 保留深色主题，规避浏览器虚拟打印机的位图渲染限制，支持中英双语版本一键导出。 Reuses the existing Playwright environment to script PDF export — forcing printBackground: true to preserve the dark theme, bypassing the bitmap-rasterisation limitations of virtual print drivers, with one-command bilingual export.
