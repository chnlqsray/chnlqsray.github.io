PROJECT · 03
GaN PL 智能分析平台
⬡ 本地部署

一套面向半导体光谱研究的 AI 增强分析工作台，将博士课题中分散的实验处理脚本整合为统一界面，覆盖从原始文件解析到 AI 辅助论文写作的完整链路。约 6,200 行代码，本人未直接编写一行。

  • 批量处理器：字节级逆向解析 SPF2 文件，自动校正、坏点修复与归一化
  • 峰拟合：能量空间多峰高斯拟合，输出峰位、半高宽、拟合优度 R²，支持导出 xlsx
  • 功率系列分析：Log-log 提取发光机制斜率 + 速率方程全局优化求解辐射寿命
  • Mapping 热图：交互式二维发光图 + 离子注入区域自动检测标注
  • 寿命对比：跨样品趋势图，可视化离子注入对载流子动力学的影响
  • AI Copilot：Python 提取物理描述符 → Groq 生成图注与 Results & Discussion 草稿

Infrastructure

基础设施 · 自动化运维

支撑以上所有部署应用持续运行的后台系统，以及每日自动推送个人投资数据的定时任务。代码分别托管于 finance_app_cloud 与 etf_daily_report 仓库。

应用保活系统

GitHub Actions 每 6 小时触发，Playwright 无头浏览器模拟真实用户访问，自动检测并点击 Streamlit 休眠唤醒按钮，含截图存档，确保两款应用全天候在线。

ETF 每日日报

工作日定时执行，调用腾讯行情 API 拉取 A 股数据（解决 GitHub Actions 海外服务器跨境限制），yfinance 拉取美股数据，计算五期收益率，生成带颜色标注的 HTML 邮件每日推送。
