PROJECT · 03
GaN PL 智能分析平台 GaN PL Intelligent Analysis Platform
⬡ 本地部署⬡ Local Deploy

一套面向半导体光谱研究的 AI 增强分析工作台，将博士课题中分散的实验处理脚本整合为统一界面，覆盖从原始文件解析到 AI 辅助论文写作的完整链路。约 6,200 行代码，本人未直接编写一行。 An AI-augmented spectral analysis workspace for semiconductor photoluminescence research, consolidating previously fragmented experimental scripts into a unified interface covering the full chain from raw file parsing to AI-assisted scientific writing. ~6,200 lines of code, not a single line written directly by me.

  • 批量处理器：字节级逆向解析 SPF2 文件，自动校正、坏点修复与归一化 Batch processor: Byte-level reverse-engineered SPF2 parsing, auto correction, dead-pixel repair and normalisation
  • 峰拟合：能量空间多峰高斯拟合，输出峰位、半高宽、拟合优度 R²，支持导出 xlsx Peak fitting: Multi-peak Gaussian fitting in energy space, outputting peak position, FWHM, R², with xlsx export
  • 功率系列分析：Log-log 提取发光机制斜率 + 速率方程全局优化求解辐射寿命 Power series analysis: Log-log slope extraction for emission mechanism identification + global rate-equation optimisation for radiative lifetime
  • Mapping 热图：交互式二维发光图 + 离子注入区域自动检测标注 Mapping heatmap: Interactive 2D photoluminescence map + automatic ion-implanted region detection
  • 寿命对比：跨样品趋势图，可视化离子注入对载流子动力学的影响 Lifetime comparison: Cross-sample trend plots visualising the effect of ion implantation on carrier dynamics
  • AI Copilot：Python 提取物理描述符 → Groq 生成图注与 Results & Discussion 草稿 AI Copilot: Python extracts physical descriptors → Groq generates figure captions and Results & Discussion drafts

Infrastructure

基础设施 · 自动化运维 Infrastructure · Automated Operations

支撑以上所有部署应用持续运行的后台系统，以及每日自动推送个人投资数据的定时任务。代码分别托管于 finance_app_cloud、etf_daily_report 与 chnlqsray.github.io 仓库。 Backend systems keeping all deployed applications running continuously, plus a scheduled task that auto-delivers daily personal investment data. Code hosted in the finance_app_cloud, etf_daily_report, and chnlqsray.github.io repositories.

应用保活系统 Keep-Alive System

GitHub Actions 每 6 小时触发，Playwright 无头浏览器模拟真实用户访问，自动检测并点击 Streamlit 休眠唤醒按钮，含截图存档，确保两款应用全天候在线。 GitHub Actions triggers every 6 hours; Playwright headless browser simulates real user visits, automatically detects and clicks the Streamlit wake-up button, with screenshot archiving to ensure both apps remain online around the clock.

ETF 每日日报 ETF Daily Report

工作日定时执行，调用腾讯行情 API 拉取 A 股数据（解决 GitHub Actions 海外服务器跨境限制），yfinance 拉取美股数据，计算五期收益率，生成带颜色标注的 HTML 邮件每日推送。 Runs on a weekday schedule, fetching A-share data via Tencent Market API (bypassing GitHub Actions cross-border restrictions) and US equity data via yfinance, computing five-period returns and delivering a colour-coded HTML email daily.

作品集 PDF 导出 Portfolio PDF Export

复用同一 Playwright 环境构建脚本化 PDF 导出方案，强制 printBackground: true 保留深色主题，规避浏览器虚拟打印机的位图渲染限制，支持中英双语版本一键导出。 Reuses the existing Playwright environment to script PDF export — forcing printBackground: true to preserve the dark theme, bypassing the bitmap-rasterisation limitations of virtual print drivers, with one-command bilingual export.
